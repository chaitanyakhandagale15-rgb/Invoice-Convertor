Add complete authentication to **InvoiceBridge** using **Clerk with Google Sign-In**.

The app is already fully built using:

* Next.js 14 (App Router)
* TypeScript
* Tailwind CSS
* Existing working backend APIs
* Database already stores invoices

Goal: add secure Google authentication and make InvoiceBridge a production-ready SaaS.

---

## Authentication Requirements

Use:

**Clerk + Google OAuth**

Implement:

* Sign up with Google
* Sign in with Google
* Protected routes
* User-specific invoice history
* Session persistence
* Redirect unauthenticated users

---

## Install Dependencies

Install:

```bash
npm install @clerk/nextjs
```

---

## Environment Variables

Add:

```env
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
```

---

## Configure Clerk

Create:

### `middleware.ts`

Protect these routes:

* `/dashboard`
* `/upload`
* `/convert`
* `/download`

Unauthenticated users should redirect to:

```plaintext
/sign-in
```

## Update Root Layout

Update:

### `src/app/layout.tsx`

Wrap app with:

`<ClerkProvider>`
